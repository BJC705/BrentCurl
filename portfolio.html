Flappy Pug

The idea of Flappy Pug was inspired by my dog Frank. I used HTML canvas and vanilla JS to draw the animations. I created multiple functions that detected if the Pug hits the bacon, or if the pug hits the ground. Along with a function the increments the score every time the pug passes the bacon.

Pacman Animation

The Pacman animation was my first time creating animations with JS and HTML Canvas. I created a custom sprite sheet that allowed me to import the different colored Pacman's onto the canvas. I created a function that randomly makes a new Pacman appear using Mathrandom.
